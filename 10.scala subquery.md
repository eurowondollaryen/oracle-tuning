# 10. 스칼라 서브쿼리

## 스칼라 서브쿼리란?
- SELECT 절의 컬럼에 들어가는 서브쿼리
- 스칼라 서브쿼리는 함수와 동일하다.(옵티마이저 관점)
- 주 쿼리에서 수행되는 데이터의 건수만큼 반복 수행된다.(캐시 사용 가능)

## 스칼라 서브쿼리와 함수의 차이점
- 동작은 동일하게 한다.
- 함수를 사용하는 경우는, 코드 반복을 줄이려 하는 것인데, SQL에서 함수를 만들어서 여러 곳에서 사용하는 것은 위험부담이 있다. (RECURSIVE CALL 발생가능), 관리포인트가 하나 늘어나는 것이다.
